# Data Engineering Portfolio & Coursework

Welcome to my Data Engineering repository. This space serves as both a portfolio showcasing my end-to-end data pipelines and a digital notebook tracking my daily coursework, syntax practice and experimental code.

## Repository Structure

### Core Projects (`/projects`)
This folder contains my standalone, end-to-end data projects. Each project has its own dedicated README detailing the architecture and tech stack.

* **[hotel_performance/](projects/hotel_performance_analysis)**: An end-to-end analytical pipeline cleaning and transforming hotel booking data using **PostgreSQL**, visualized via a **Power BI** dashboard to uncover operational insights.
* **[json_to_csv/](projects/json_to_csv)**: A data pipeline utilizing **Python** and **Pandas** to extract, parse, and flatten deeply nested JSON data from web endpoints (DummyJSON API) into analysis-ready CSVs.
* **[news_etl/](projects/news_etl)**: An ETL pipeline extracting live data from the News API and loading it into a storage database.
* **[gas_prices_etl/](projects/gas_prices_etl)**: A custom ETL pipeline extracting, transforming and loading gas price data.


### Coursework & Practice
The following folders act as my digital scratchpad :
* **[/SQL](SQL)**: Scripts focusing on query practice, assignments (e.g., Sales Inventory analysis), and window functions.
* **[/python](python)**: Core programming concepts, loops, functions, and data structures.
